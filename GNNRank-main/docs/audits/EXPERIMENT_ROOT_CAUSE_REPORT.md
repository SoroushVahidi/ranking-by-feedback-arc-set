# Root-Cause Report: Manuscript-vs-Repository Experiment Contradictions

## Confirmed causes
- **Stale aggregate artifact used for classical values:** `all_methods_metrics.csv` does not agree with raw result-array aggregates.
- **Method-label swap in stale path:** btl and davidScore aggregate profiles are swapped in stale artifact consumption.
- **Stale denominator assumption:** some manuscript-facing summaries used 80 instead of canonical 81 datasets.

## Non-causes
- OURS values are consistent with canonical result-array evidence.
- Runtime/speedup/Pareto claims remain supported by runtime fields in result-array evidence.

## Corrective action
Canonical outputs now live under:
- `outputs/audits/`
- `outputs/paper_tables/`
and are regenerated by `scripts/paper/rebuild_experiment_tables.py` with automated checks in `scripts/paper/validate_experiment_tables.py`.
